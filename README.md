# Digital menu managrment with qr code inside the resturant & cafe

#Overview
This project presents a simple digital menu management system designed for restaurants and cafes. Customers can scan a QR code inside the restaurant to access the menu directly on their mobile devices without needing a printed menu.

#Features
- QR code access to the digital menu  
- Easy menu browsing for customers  
- Contactless ordering experience  
- Simple management of menu items  

#Purpose
The goal of this project is to demonstrate how digital tools and QR code technology can improve customer experience and simplify menu management inside restaurants and cafes.

#Technology
- Web technologies  
- QR code system  
- Simple digital menu interface
